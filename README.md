# crow

## Setup development environemnt

### NodeJS and NPM 

```bash
apt install nodejs npm 
```

### Clone and Install

```bash
git clone git@github.com:vorder/crow.git
cd crow
npm install
```

### Hot reload

```bash
npm run serve
```

### Tests

```bash
npm test
```

### Deploy

```bash
npm run build
```

### Lint

```bash
npm run lint
```
